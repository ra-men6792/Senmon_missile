タイトル：誘導ミサイルシュミレーション『MissileParty』

概要：
4つの種類の誘導ミサイルを撃つことができる。
モードを切り替えて、様々なパターンで撃てる。
以下、4つの種類のミサイル紹介
Line：直線的に敵に向かってくる
RandLine：敵からずれた位置に着弾する。
Curve：揺らぎながら曲線的に敵に向かってくる。
RandCurve：揺らぎながら敵からずれた位置に着弾する。

操作説明：

自機(赤)
/////////////////////////////////////////////
  十字キー：移動
  Spaceキー：ミサイル発射(3〜10発)
　1キー：通常　4種ランダムでミサイルを打つ
　2キー：Lineのミサイルのみを撃つ
　3キー：Curveのミサイルのみを撃つ
　4キー：RandLineのみを撃つ
　5キー：RandCuraveのみを撃つ
　6キー：Line、RandLineをランダムで撃つ
　7キー：Curave、RandCurveをランダムで撃つ
/////////////////////////////////////////////

敵(シアン)
/////////////////////////////////////////////
左クリック：敵の座標を変更（クリックした位置に）
/////////////////////////////////////////////

Rキー：リセット　軌跡も消える
右クリック：ビュー角度変更
マウスホイール：拡大縮小
